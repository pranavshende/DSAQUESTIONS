# 📘 DSA LeetCode Top Questions

This project provides a curated list of **20 important DSA questions for each topic**, organized to help with structured practice and efficient interview preparation.

---

## 🚀 Topics Covered

Arrays Strings LinkedList Stack/Queue Trees Graphs BinarySearch DynamicProgramming Heap Backtracking Tries SlidingWindow TwoPointers BitManipulation Math

---

## 📂 Features

- 20 questions for each topic  
- Covers Easy, Medium, and Hard levels  
- Topic-wise structured preparation  
- Direct links to LeetCode problems
